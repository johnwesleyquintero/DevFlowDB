## Quick Start

1. Clone the repository:
   ```bash
   # Clone repository
   $ git clone https://github.com/your-username/devflowdb.git
   $ cd devflowdb
   ```

2. Install dependencies:
   ```bash
   # Install dependencies
   $ npm install
   ```

3. Start the development server:
   ```bash
   # Start development server
   $ npm run dev
   ```
